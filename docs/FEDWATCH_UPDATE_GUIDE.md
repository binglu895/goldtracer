# CME FedWatch 数据更正指南

## 现状
由于 CME 官网的强力反爬虫机制，自动抓取目前不可用。系统默认显示由 AI 检索或硬编码的权威数据。

## 如何手动更正数据

您可以直接在 Goldtracer UI 界面进行更正：

1. **点击验证链接**：在 FedWatch 卡片右上角点击 `CME数据↗`。
2. **确认数值**：在 CME 官网查看“Target Rate Probabilities”。
3. **执行更正**：
   - 点击卡片右上角的 `修正` 按钮。
   - 在弹出的输入框中分别输入最新的“维持利率”和“降息”概率。
   - 点击确认，数据将通过 `/api/admin/fedwatch/update` 接口永久保存至数据库。

## 显示说明
- **倒计时**：系统自动计算至 2026-03-18 会议的剩余时间。
- **上次更新**：卡片底部会显示数据的最后核验时间。
- **Source**：标记为 "CME"，代表数据源自芝加哥商品交易所。

## 开发说明
手动更正接口：`POST /api/admin/fedwatch/update`
负载示例：
```json
{
  "prob_pause": 84.1,
  "prob_cut_25": 15.9,
  "meeting_date": "2026-03-18"
}
```
该接口会自动计算隐含利率并更新今日的 `daily_strategy_log` 记录。
